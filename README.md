# PromptBuilder (Prototype)

Web-прототип конструктора промптов. Пользователь выбирает формат результата, индустрию, экспертов и дополнительные поля, после чего получает готовый промпт для копирования или скачивания.

## Технологии

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Docker (production deploy)

## Функции прототипа

- Генерация промпта на основе конфигурации (индустрии, эксперты, форматы, поля)
- Админ-страница для редактирования конфигурации
- Копирование результата в буфер
- Экспорт результата: TXT, Markdown, HTML

## Переменные окружения

Пример файла переменных: `.env.example`

Рекомендуемые переменные:

- `NODE_ENV=production`
- `PORT=3000`
- `NEXT_TELEMETRY_DISABLED=1`

Создай файл `.env.local` для локального запуска или используй `--env-file` при запуске контейнера.
