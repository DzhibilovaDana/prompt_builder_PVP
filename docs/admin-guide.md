# Руководство администратора

1. Авторизуйтесь под учётной записью с `is_admin=true`.
2. Перейдите в административный интерфейс.
3. Редактируйте справочники и системную конфигурацию через API/формы.
4. Изменение `/api/config` доступно только администратору.
5. Проверяйте корректность справочников перед публикацией изменений.
