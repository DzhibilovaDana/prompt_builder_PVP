# Эксплуатационная инструкция

## Запуск
- Используйте Docker Compose/ Dockerfile проекта.
- Контейнер веб-приложения: Next.js.
- Контейнер/сервис БД: PostgreSQL.

## Переменные окружения
- `DATABASE_URL` — строка подключения PostgreSQL.
- `NODE_ENV` — режим окружения.
- Прочие переменные задаются в `.env`/секретах деплоя.

## Инициализация схемы
- Выполните init-скрипты/миграции (`npm run db:init`).
- Проверьте наличие таблиц users/workspaces/prompts/prompt_versions.

## Резервное копирование
- Прототип: `pg_dump` для полной выгрузки БД.
- Восстановление: `psql`/`pg_restore` в целевую БД.
- Регламент: ежедневный backup + проверка тестового восстановления.
