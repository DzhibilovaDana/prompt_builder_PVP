# Руководство пользователя

1. Войдите в систему.
2. Откройте конструктор prompt.
3. Заполните поля (шаблон, роль эксперта, отрасль, формат).
4. Нажмите сохранение — prompt попадёт в каталог.
5. Используйте поиск/фильтры по категории, тегам и тексту.
6. Откройте карточку prompt для просмотра истории версий.
7. Выполните экспорт в поддерживаемый формат (txt/md/html).
