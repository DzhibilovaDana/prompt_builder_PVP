# Глоссарий

- **Prompt** — текстовая инструкция для LLM, сохраняемая как сущность `prompt`.
- **Template** — шаблон структуры prompt с предзаполненными блоками.
- **Component** — переиспользуемый логический блок конструктора (селектор, поле, секция).
- **Format** — формат вывода или структура ответа (например, список, Markdown, HTML).
- **Industry** — отраслевая специализация для фильтрации и настройки prompt.
- **Expert role** — роль эксперта (аналитик, маркетолог и т.д.) как параметр генерации.
- **Workspace** — рабочее пространство с участниками и ролевым доступом.
- **Version** — зафиксированная редакция prompt в истории `prompt_versions`.
- **Metadata** — JSON-поле с дополнительными параметрами prompt.
- **Tag** — короткая метка для поиска и категоризации.
- **Category** — основная классификация prompt по тематике.
